# Coveo Java SDK

This SDK makes it simple to push documents to a Coveo Push Connector endpoint for Coveo. 

## Getting Started

This repository is used in a variety of MC+A Java projects to push content to Coveo Push Endpoints.

## Requirements

- Java +8
- Gradle

## Build

```
gradle build
```
